# Forms in HTML
-> Forms are used in HTML to collect data from the user through multiple inputs
-> It is created using <table> tag.

# Tags mainly used in Forms
-> <form> : This tag is used to group input fields together so that the user's data can be collected and submitted
-> <fieldset> : This tag is used to group related form elements together inside a form by creating an outline
-> <legend> : This tag is used to give a caption or title for the form over the <fieldset>
-> <label> : This tag is used to provide a descriptive text for an input fields
-> <input> : This tag is used to create an input field where users can enter or select data


# Attributes in Forms
-> for : It is used to create a connection between the label and input
-> name : It tells the browser that the radio buttons belong to the same group
-> label : It is used to give a title for the classification given for the optgroup

# Input types in Forms
-> It is used to specify the type of the input expected from the user
-> It is given as an attribute in the <input> tag

# types
-> text : It directly accept plain text of any type of characters
-> email : It takes the input in the format of an email 
-> number : It takes input in numerical values
-> submit : It submits the all the data given by the user
-> reset : It clears all the existing data entered by the user
-> password : It hides the characters entered by using dots or asterisks
-> checkbox : It allows the user to select multiple choices from the given options
-> radio : It allows user to select a single choice from the given options



# Dropdown
-> Dropdown is an HTML Form control which allows the user to select choices from the given list
-> <select> : It is used to create a Dropdown
-> <option> : It is used to add options to the Dropdown
-> <optgroup> : It is used to give a classification for the dropdown options
